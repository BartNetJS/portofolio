# When AI Agents Break Your App

This folder contains the blog post sources and rendered variants.

- `post.md` — Markdown source for the article. Used by `index.html` via `markdown-mermaid.js`.
- `index.html` — Markdown-rendered entry that loads `post.md`.
- `full.html` — Full standalone HTML copy (for cases where Markdown is insufficient).

To preview:

```bash
python -m http.server 8000
# then open http://localhost:8000/Blogs/when-ai-agent-breaks-your-app/
```
