üìñ Getting Started

Prerequisites

  • .NET 9 SDK or later
  • Visual Studio Code or Visual Studio 2022+
  • Azure Storage Emulator (or Azurite)
  • Azure OpenAI API keys (optional, supports local models)

Local Development

  • Clone the repository
  • Run dotnet restore
  • Start storage emulator: ./start-storage-emulator.ps1
  • Run Aspire dashboard: dotnet run --project AIPlayground.AppHost
  • Access dashboard at http://localhost:18888
